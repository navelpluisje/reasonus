Control Panel

The heart of ReaSonus Native is the ReaSonus Control Panel. This is where you can assign REAPER actions to the function keys, manage your plugin mappings or create custom filters. Also all the settings are available in the settings tab.

Not all the options are available for all the devices. For the FaderPort V2 there is no plugin mapping and custom filter available.
