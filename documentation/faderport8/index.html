Fader Modes

The Faderport has multiple fader modes. These modes control the functionality of the fader, mute, solo, select button and the content of the displays.

By pressing the Plugin, Send and Pan (which shows the receives) when they are already active, you can switch between Hui and Track mode

  1. Display:
    • Line 1: Track name
    • Line 2:
      • Default: Track number
      • Left Shift: Track number
      • Right Shift: Track number
      • Arm: Input name
    • Line 3:
      • Default: Pan value (depending on the mode it is pan or width)
      • Left Shift: Pan value 1
      • Right Shift: Pan value 1
      • Arm: Track monitor mode
    • Line 4:
      • Default: Track number
      • Left Shift: Pan value 2
      • Right Shift: Pan value 2
      • Arm: Recording mode
  2. Meter: Shows the Vu meter only in default mode. The meter is hidden when using the shift and or arm button.
  3. Value bar:
    • Default: Displays the Tracks current pan position.
    • Left Shift: Track volume.
  4. Pan/Param:
    • Default: Controls pan value 1 for the currently selected track. Push to set the pan position back to center.
    • Left Shift: Controls pan value 1 for the currently selected track. Push to set the pan position back to center.
  5. Select button:
    • Default: Unique-select the Track. Other Tracks will be de-selected.
    • Left Shift: Select the Track. All previous selected Tracks will also stay selected.
    • right Shift: Selects the range of Tracks between the previous selected Track and the current selected Track.
    • Arm: Toggle Track arm.
  6. Solo: Isolates the corresponding Tracks output signal in the mix. Press and hold to solo for a couple of seconds.
  7. Mute: Mutes the corresponding Tracks output signal. Press and hold to mute for a couple of seconds.
  8. Touch-Sensitive Fader:
    • Default: Control volume levels.
    • Left Shift: Control the pan value 1.
    • right Shift: Control the pan value 2.

Transport

  1. Stop: Stops playback.
  2. Loop: Engages / disengages Looping
  3. Play / Pause: Starts playback at the current playback- cursor position. Press again to pause playback.
  4. Rewind:
    • Default: Rewinds in a slow pace. You can toggle the speed by pressing the button. All the other keys will stop the rewind
    • [Left Shift]: Go to the start of the project
  5. Fast Forward:
    • Default: Fast Forwards in a slow pace. You can toggle the speed by pressing the button. All the other keys will stop the Fast Forward
    • [Left Shift]: Go to the end of the project
  6. Record: Press to start recording at the current playback-cursor position for record-enabled tracks.

Automation

  1. Latch:
    • Default: Engages Latch Automation on currently selected track.
    • Left Shift: Save your session.
    • Left Shift: Engages Latch Automation globally.
  2. Trim:
    • Default: Engages Trim Automation on currently selected track.
    • Left Shift: Redo the last action.
    • Left Shift: Engages Trim Automation globally.
  3. Off:
    • Default: Engages Latch Preview Automation on currently selected track.
    • Left Shift: Undo the last action.
    • Left Shift: Engages Latch Preview Automation globally.
  4. Read:
    • Default: Engages Read Automation on currently selected track.
    • Left Shift: Not implemented yet.
    • Left Shift: Engages Read Automation globally.
  5. Write:
    • Default: Engages Write Automation on currently selected track.
    • Left Shift: Not implemented yet.
    • Left Shift: Engages Write Automation globally.
  6. Touch:
    • Default: Engages Touch Automation on currently selected track.
    • Left Shift: Not implemented yet.
    • Left Shift: Engages Touch Automation globally.

General Controls

  1. Encoder: The behaviour differs per fader mode
    • Track mode: Rotating the encoder will alter the track pan
    • Plugin mode: When in Hui mode this will scroll through the track plugins
    • Send mode: When in Hui mode this will scroll through the track sends
    • Receive mode: When in Hui mode this will scroll through the track receives
  2. Encoder Click:
    • Default: Toggle between pan values (left/right, pan/width, etc), depending on the track pan mode
    • Left Shift: Reset the pan and with to default values
  3. Arm:
    • Default: Behaves as a modifier key. Pressing this button will enable you to set ther record arm with the track Select buttons. A short press of this button will maintain the pressed state
    • Left Shift: Arm all tracks for recording
    • Right Shift: Unarm all tracks for recording
  4. Solo Clear: Unsolo all the tracks soloed tracks
  5. Mute Clear: Unmute all the muted tracks
  6. ByPass:
    • Default: Bypass all plugins for the selected tracks
    • Left Shift: Bypass all plugins for all the tracks
  7. Macro:
    • Default: Toggle the window filters dialog for managing the acttions attached to the function keys
    • Left Shift: Toggle the window for managing the filters
  8. Link:
    • Default:
      • When not controlling a plugin, it will make the 8th fader controll the last touched parameter. The group of actual track faders will beb 7 from then.
      • When controlling a plugin, it will toggle the plugin map edit mode. You are then able to edit your plugin mapping
    • Left Shift: Toggle following the cursor

Mix Management

Mix management houses a serie of filters to filter the tracks to ease your mixing even more. Here a description of all the filter options available

  1. Audio: Filter all theh tracks that have audio items in them.
  2. VI:
    • Default: Filter all tracks with Virtual Instruments
    • Left Shift: Filter all tracks with MIDI items
  3. Bus:
    • Default: Show only the top level tracks
    • Left Shift: Filter all the tracks with hardware outputs
    • Right Shift: Filter all the folder tracks
  4. VCA:
    • Default: Filter all the VCA tracks
    • Left Shift: Filter all the tracks with effects attached to them
    • Right Shift: Filter all the tracks with receives
  5. All:
    • Default: Reset all the filter and show all the tracks
    • Left Shift: Show the custom menu's. This will display all your fiolters in the track displays. Use the select button to select the filter
    • Right Shift: Filter all the tracks with sends

Session Navigator

The Session Navigator provides quick navigation and session controls. Each button alters the functions of the push-button encoder and the Next and Prev buttons on either side.

  1. Channel: Control channel scrolling
    • Previous:
      • Default: Moves the faderbank by steps of 1 track
      • Left Shift Moves the faderbank by steps of 8/16 tracks
    • Encoder:
      • Default: Moves the faderbank by steps of 1 track
      • Left Shift Moves the faderbank by steps of 8/16 tracks
    • Encoder Click: Not implemented
    • Next:
      • Default: Moves the faderbank by steps of 8/16 tracks
      • Left Shift Moves the faderbank by steps of 1 track
  2. Master: Control the master channel
    • Previous:
    • Encoder:
    • Encoder Click:
    • Next:
  3. Zoom: Handle zooming the navigator in Reaper
    • Previous:
      • Default: Zoom Reaper out horizontal
      • Left Shift Zoom Reaper out vertical
    • Encoder:
      • Default: Zoom Reaper in/out horizontal
      • Left Shift Zoom Reaper in/out vertical
      • Default: Toggle track zoom to minimum height
      • Left Shift Toggle track zoom to maximum height
    • Encoder Click:
    • Next:
      • Default: Zoom Reaper in horizontal
      • Left Shift Zoom Reaper in vertical
  4. Click: Handle the different metronome functions
    • Previous: Set metronome speed to 1x
    • Encoder: Adjust the Metronome volume
    • Encoder Click:
      • Default: Toggle metronome on/off
      • Left Shift Show metronome/pre-roll settings
    • Next: Set metronome speed to 2x
  5. Scroll: Handle scrolling the navigator in Reaper
    • Previous:
      • Default: Scroll Reaper up
      • Left Shift Scroll Reaper left
    • Encoder:
      • Default: Scroll Reaper up/down
      • Left Shift Scroll Reaper left/right
    • Encoder Click:
      • Default: Vertical scroll selected tracks into view
      • Left Shift Horizontal scroll to put play cursor at 50%
    • Next:
      • Default: Scroll Reaper down
      • Left Shift Scroll Reaper right
  6. Section: Handle adding, deleting and editing regions
    • Previous: Filter all tracks with Virtual Instruments
      • Default: Goto/select previous marker/region
      • Left Shift Set the start point for a new section
    • Encoder:
      • Default: Move edit cursor forward/backward one beat
      • Left Shift Move cursor left/right one pixel
      • Right Shift Move cursor left/right, creating time selection
    • Encoder Click:
      • Default: Edit region near cursor
      • Left Shift Move edit cursor to start of current measure
      • Right Shift Delete region near cursor
    • Next:
      • Default: Goto/select next marker/region
      • Left Shift Set the end point for a new section
  7. Bank: Control bank scrolling. Depending on your controller with 8 or 16 channels
    • Previous:
      • Default: Moves the faderbank by steps of 8/16 tracks
      • Left Shift Moves the faderbank by steps of 1 track
    • Encoder:
      • Default: Moves the faderbank by steps of 8/16 tracks
      • Left Shift Moves the faderbank by steps of 1 track
    • Encoder Click: Not implemented
    • Next:
      • Default: Moves the faderbank by steps of 8/16 tracks
      • Left Shift Moves the faderbank by steps of 1 track
  8. Marker: Handle adding, deleting and editing markers
    • Previous: Goto/select previous marker/region
    • Encoder:
      • Default: Move edit cursor forward/backward one beat
      • Left Shift Move cursor left/right one pixel
    • Encoder Click:
      • Default: Insert marker at current position
      • Left Shift Insert and/or edit marker at current position
      • Right Shift Delete marker near cursor
    • Next: Goto/select next marker/region

Function Controls

Each of the eight buttons also gives you acces to configurable functions by pressing the Left Shift buttons together with these buttons

  • F1: Open or close the Mixer
  • F2: Open or close the Media Explorer
  • F3: Open or close the Routing Matrix
  • F4: Open or close the Effects Browser
  • F5: Open or close the Regions and Markers window
  • F6: Open or close the Project Media browser
  • F7: Open or close the Big Clock
  • F8: Open or close the track Manager
