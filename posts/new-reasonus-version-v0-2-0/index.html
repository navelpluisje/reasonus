New ReaSonus version (v0.2.0)

A new version of ReaSonus Native has been released.

What's Changed

All devices

  • Better dialog handling via actions. Show meassage when dialog not supported for installed device

FaderPort 8/16

  • Added Master Fader Mode with plugin control. The Master button makes the last track control the master track.
  • Added Master Fader Automation
  • Fixed issue with wrong assigned solo buttons on FP16

FaderPort V2

  • Added separate action for functions dialog
  • Disable fader and controls when no track selected
  • Make all the track buttons work on master track as well
  • Added option to select and control hidden selected tracks

Install

ReaSonus Native is now available in ReaPack. There is no need to add a repository, but works out-of-the-box. In ReaPack search for ReaSonus and you can install it.

Documentation

Documentation can be found here. If you see typo's or other issues with the documentation, please report them, so the documentation can get updated.

Thanx to anyone reporting issues and help me debug them.

Full Changelog: https://github.com/navelpluisje/Reasonus-Native/compare/0.1.4...0.2.0
