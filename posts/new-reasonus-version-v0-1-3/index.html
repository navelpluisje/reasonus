New ReaSonus version (v0.1.3)

A new version of ReaSonus Native has been release.

What's Changed

All devices

  • Fixed issue with resetted function ids on Reaper startup
  • Quite some pan fixed for all devices and both master and track channels

FaderPort 8/16

  • Display fix for the FaderPort 16
  • Fixed issue where Reaper could crash when editting a non-existing filter
  • Improved the behaviour of Parent Select for the filters
  • Fixed filter key issue when adding filters too quick
  • Added version number to the menu
  • Unselect all tracks before solo-select
  • Fixed issue with time code when ruler was selected.
  • Now displays time code when nb of trachs is minimal

FaderPort V2

  • Set next visible item as selected when selected track is not visible
  • Some V2 session improvements

Install

ReaSonus Native is now available in ReaPack. There is no need to add a repository, but works out-of-the-box. In ReaPack search for ReaSonus and you can install it.

Documentation

Documentation can be found here. If you see typo's or other issues with the documentation, please report them, so the documentation can get updated.

Thanx to anyone reporting issues and help me debug them.

Full Changelog: https://github.com/navelpluisje/Reasonus-Native/compare/v0.1.1...0.1.3
